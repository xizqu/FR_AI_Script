1. When entering new commands & defines, follow the same format. Make sure the commands/defines file also has a blank space at the top.

2. If you get build errors such as "Error in defines file: "pokemon.aisp" on line 1.", ignore it (line 1 can sometimes be buggy).

3. The script compiles from a file titled "AI_Script.ais".

4. Install python 3.6 or higher, and double click the compiler to run it.

5. The file will not compile until it is error-free.

6. The output file contains a string of bytes to paste into the hex editor at your desired offset. Make sure it matches the offset of the #dynamic in your script.
